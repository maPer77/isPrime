# isPrime

Check if number is prime

# Example

https://maper77.github.io/isPrime/

# Use

````
<script type="module">
    import isPrime from "./index.js";

    const [ result, list ] = isPrime(17);

    console.log(result, list);
</script>
````

````
result: true
list: [2, 3, 5, 7, 11, 13, 17]
````
